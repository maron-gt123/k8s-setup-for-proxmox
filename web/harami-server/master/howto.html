はらむはらみサブ限サーバの入り方ガイド

Java版・統合版どちらでも簡単に接続できます。
入室にははらむはらみTikTokのサブスク入会が必要です。

Java版 (PC)

  1. Minecraft launcherを起動 → 「Java EDITION」を選択 → 「プレイ」
  2. Minecraftが起動 → 「マルチプレイ」
  3. 「サーバーを追加」をクリック
  4. サーバー名：任意 (例：はらむはらみサブスク限定鯖)
  5. サーバーアドレス：mc.example.com
  6. 表示されれば接続成功
トップに戻る

統合版 (PC / モバイル)

  1. Minecraft launcherを起動 → 「BEDROCK EDITION」を選択 → 「プレイ」
  2. Minecraftが起動 → 「プレイ」 → 「サーバー」 → 「+ サーバー追加」
  3. サーバー名：任意 (例：はらむはらみサブスク限定鯖)
  4. サーバーアドレス：mc.example.com または 111.111.111.111
  5. ポート番号：19132
  6. サーバーを追加 → 「他のサーバー」に追加確認

サブスク認証

  • 認証条件
    • TikTokサブスク入会済み
    • サブスク限定Discordに参加
    • Minecraft接続時に発行されるワンタイムパスワードで認証
  • 注意点
    • サブスク解除後、再ログインはできなくなります
    • ワンタイムパスワード認証はBOTへのDMで実施
    • 定期的な再認証があります
  • 認証方法
    • Java版：「サーバーに接続」を選択、統合版：「プレイ」を選択
    • 「このサーバをプレイするにはDiscordアカウントのリンクが必要です」と表示される
    • 「次のコードをDMで送信してください：#****」と表示 ※****はワンタイムパスワード(4桁)
    • Discord-BOT(haramis-mc-BOT)にワンタイムパスワードをDM送付
    • BOTは同Discordチャンネルの「入退室履歴」で確認可能
    • 認証後、再度ログインを実施
    • 「そのようなコードは知りません。」と返信の場合は再発行
