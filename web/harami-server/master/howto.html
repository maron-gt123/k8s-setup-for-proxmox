はらむはらみサブ限サーバの入り方

Java版・統合版どちらでも簡単に接続できます。
入室にははらむはらみTikTokのサブスク入会が必要です。

目次

各利用ハードごとのサーバー接続手順を確認しログイン認証を実施してください。
X-BOX及びSwitchでの入室は要望があれば実施いたします。

サブスク認証

Java版 (PC)

  1. Minecraft launcherを起動 → 「Java EDITION」を選択 → 「プレイ」
  2. 「マルチプレイ」
  3. 「サーバーを追加」をクリック
  4. サーバー名：任意 (例：はらむはらみサブスク限定鯖)
  5. サーバーアドレス：mc.example.com
  6. 表示されれば接続成功

統合版 (PC / モバイル)

  1. Minecraft launcherを起動 → 「BEDROCK EDITION」を選択 → 「プレイ」
  2. 「プレイ」 → 「サーバー」 → 「+ サーバー追加」
  3. サーバー名：任意 (例：はらむはらみサブスク限定鯖)
  4. サーバーアドレス：mc.example.com または 111.111.111.111
  5. ポート番号：19132
  6. サーバーを追加 → 「他のサーバー」に追加確認

サブスク認証

サブスクメンバー認証のため、ワンタイムパスワード認証が必要です。

認証条件

  • TikTokサブスク入会済み
  • サブスク限定Discordに参加
  • マインクラフト接続時に発行されるワンタイムパスワードで認証

注意点

  • サブスク解除後、再ログインはできなくなります
  • ワンタイムパスワード認証はBOTへのDMで実施
  • 定期的な再認証があります

認証方法

  1. Java版：「サーバーに接続」 統合版：「プレイ」を選択
  2. 「このサーバをプレイするにはDiscordアカウントのリンクが必要です」と表示
  3. 「次のコードをDMで送信してください：#****」と表示（****はワンタイムパスワード4桁）
  4. Discord BOT (haramis-mc-BOT) にワンタイムパスワードをDM送信
  5. BOTは同Discordチャンネルの「入退室履歴」で確認可能
  6. 「あなたのDiscordアカウントは [ユーザ名] とリンクしました」と返信が来る
  7. 認証完了後、再度ログイン可能
