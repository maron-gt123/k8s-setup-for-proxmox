はらむはらみサブ限サーバの入り方ガイド

Java版・統合版どちらでも簡単に接続できます。
入室にははらむはらみTikTokのサブスク入会が必要です。

目次

各利用ハードごとのサーバー接続手順を確認しログイン認証を実施してください
X-BOX及びSwichでの入室は要望があれば実施いたします

  • Java版 (PC)
  • 統合版 (PC / モバイル)
  • 統合版 (PS5)
  • サブスク認証

Java版 (PC)

  1. Minecraft launcherを起動 → 「Java EDITION」を選択 → 「プレイ」
  2. Minecraftが起動 → 「マルチプレイ」
  3. 「サーバーを追加」をクリック
  4. サーバー名：任意 (例：はらむはらみサブスク限定鯖)
  5. サーバーアドレス：mc.example.com
  6. 表示されれば接続成功

統合版 (PC / モバイル)

  1. Minecraft launcherを起動 → 「BEDROCK EDITION」を選択 → 「プレイ」
  2. 「プレイ」 → 「サーバー」 → 「+ サーバー追加」
  3. サーバー名：任意 (例：はらむはらみサブスク限定鯖)
  4. サーバーアドレス：mc.example.com または 111.111.111.111
  5. ポート番号：19132
  6. サーバーを追加 → 「他のサーバー」に追加確認

統合版 (PS5)

  1. 例：サーバーアドレス入力 → ポート番号設定 → 接続確認

サブスク認証

サブスクメンバー認証のため、ワンタイムパスワード認証が必要です

  • 認証条件
    • TikTokサブスク入会済み
    • サブスク限定Discordに参加
    • マインクラフト接続時に発行されるワンタイムパスワードで認証
  • 注意点
    • サブスク解除後、再ログインはできなくなります
    • ワンタイムパスワード認証はBOTへのDMで実施
    • 定期的な再認証があります
  • 認証方法
    • Java版：「サーバーに接続」を選択　統合版：「プレイ」を選択
    • 「このサーバをプレイするにはDiscordアカウントのリンクが必要です」と表示
    • 「次のコードをDMで送信してください：#****」と表示（****＝ワンタイムパスワード）
    • Discord BOT (haramis-mc-BOT) にDM送信
    • 送信後「アカウントがリンクされました」と返信される
    • エラー時は再度ワンタイムパスワード発行
