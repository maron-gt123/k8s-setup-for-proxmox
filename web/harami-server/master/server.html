サーバー概要

配信者はらむはらみと視聴者で協力や建築、イベントを楽しめるMinecraft環境を提供しています。

  • 動作確認済みversion
    • Java版：1.21.8
    • 統合版：1.21.100
  • ワールド構成
    • lobby：各ワールド移動のための中継ポイント
    • S1：メインワールド
    • S2以降：将来拡張
    • prison：BAN対象者移送先
  • 運用時間
    • 24時間運用
    • 毎朝4:00に自動再起動処理実行

ルール

  • 荒らしや不正行為は禁止
  • 他プレイヤーへの嫌がらせは禁止
  • サーバー内でのマナーを守ること
  • イベント開催時は運営の指示に従うこと

サーバー仕様

  • ワールド構成:サバイバル
  • Minecraft統合版対応 (PC / モバイル)
  • バージョン: 動作確認済みversionより過去については入室可能です
  • 各ワールドに専用ミニマップ (BlueMap) を提供

運営紹介

運営: はらむはらみ
